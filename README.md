EduManager is a web-based Student Management System designed to manage student admissions, courses, batches, and fee collection in an educational institute. The system provides an easy-to-use dashboard for administrators to track student data, course activity, and revenue in one place.

🚀 Features

📊 Dashboard Overview

Total Students

Active Courses

Pending Fees

Revenue Summary

👩‍🎓 Student Management

Add, edit, and manage student records

Store student details and email information

📝 Admission Management

Register new student admissions

Upload and verify ID documents

Track admission status

📚 Course Management

Add and manage courses

Assign courses to students

👥 Batch Management

Organize students into batches

Manage batch schedules

💰 Fee Management

Track collected and pending fees

Manage student payment records

📈 Reports

View admission and fee reports

🛠️ Technologies Used

Frontend: HTML, CSS, Bootstrap, JavaScript

Backend: PHP

Database: MySQL

Server: XAMPP / Apache
